# Typing Master Pro

A modern typing practice website built using **HTML, CSS, and JavaScript** with real-time typing statistics, paragraph customization, and local storage support.

## Features

### Typing Test

* Real-time typing speed (WPM)
* Real-time accuracy calculation
* Character-by-character highlighting
* Correct character highlighting
* Incorrect character highlighting
* Current character indicator
* Automatic paragraph completion detection

### Paragraph System

* Random paragraph generation
* Multiple paragraph support
* Select 1 to 10 paragraphs
* Meaningful sentence-based content
* Automatic paragraph progression

### Character Filters

* Small letters
* Capital letters
* Numbers
* Symbols

### Statistics

* Current WPM
* Current Accuracy
* Paragraph Progress
* Typing Status
* Average WPM
* Average Accuracy
* Best WPM
* Best Accuracy

### Pause System

* Automatic pause after 5 seconds of inactivity
* Resume from the exact position
* Pause overlay notification

### Local Storage

* Stores best WPM
* Stores best Accuracy
* Persists data after browser refresh

### User Interface

* Dark theme design
* Responsive layout
* Virtual keyboard
* Retry button
* Result modal
* Mobile-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API

## How to Run

1. Download all project files.
2. Place them in the same folder.
3. Open `index.html` in your browser.
4. Click **Start Typing**.
5. Begin practicing.
